# FoodieK Media Mapping Document

This document is an analysis-first content mapping for the current media assets in the project. No application code was modified.

## Method note

- The video files are stock-style clips with numeric filenames and no embedded metadata, so the mapping below is an inferred, production-ready content assignment based on visual style, food type, cuisine cues, and the intended app experience.
- Where the same dish appears in multiple videos, it has been distributed across different restaurants so each restaurant can own its own version of the recipe.
- The restaurant images were assigned to logo/banner/dish uses based on their visual fit and the restaurant themes in this catalog.

## Suggested AI chef portrait brief

Create 8 portrait assets for the food partners using the following brief:

- Subject: restaurant owner or chef in a kitchen setting
- Style: warm, natural, lived-in, not corporate
- Clothing: apron, chef whites, simple kitchen attire
- Mood: confident, welcoming, slightly energetic
- Lighting: natural window light or warm kitchen glow
- Background: kitchen steam, cooking station, fresh ingredients
- Avoid: business suits, stiff poses, overly polished corporate styling
